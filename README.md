# EcoLix+

🌱 Nutrimos la tierra, reducimos tus costos 🌿

## Descripción

EcoLix+ es un proyecto de emprendimiento estudiantil desarrollado en la Unidad Educativa General Eloy Alfaro, enfocado en la producción de lixiviado ecológico, un fertilizante natural que beneficia la agricultura sostenible.

## Características

- 🌿 Fertilizante 100% natural y ecológico
- 🌱 Producción sostenible
- 💚 Amigable con el medio ambiente
- 💪 Proyecto estudiantil comprometido

## Tecnologías

- React
- Vite
- Material-UI
- React Router

## Desarrollo

```bash
npm install
npm run dev
```
